# CyberFiction Website

This project is a simple website named "CyberFiction" that aims to showcase a decentralized community focusing on creating new values and profits through play in the virtual world or metaverse. The website is designed with a futuristic and cyberpunk aesthetic, emphasizing the theme of fun, collaboration, and building a positive virtual environment.

## Introduction

The website consists of several pages that provide information about CyberFiction and its core values. The main page features a captivating headline and a brief description of CyberFiction's goals. Users are encouraged to scroll down to explore more content.

## Features

- **Scroll Animation:** The website incorporates scroll animations using [Locomotive Scroll](https://github.com/locomotivemtl/locomotive-scroll) for a smooth scrolling experience.
- **Typography and Design:** The website employs a futuristic and cyberpunk design with carefully chosen typography to enhance the thematic elements.

## Usage

To view the website, simply open the `index.html` file in a web browser. Scroll down to explore different sections, and click the "JANUARY,2024" button in the navigation bar to navigate to other pages.

## Dependencies

The project utilizes the following external dependencies:

- [Locomotive Scroll](https://github.com/locomotivemtl/locomotive-scroll): A scroll animation library.
- [GSAP (GreenSock Animation Platform)](https://greensock.com/gsap/): A JavaScript animation library.

## Installation

To set up the project locally, follow these steps:

1. Clone the repository: `git clone https://github.com/your-username/cyberfiction.git`
2. Open the project folder: `cd cyberfiction`
3. Open `index.html` in a web browser.
